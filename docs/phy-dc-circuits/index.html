Concept 1

欧姆定律：电路的基础方程

已经知道：你已知道电压、电流、电阻这三个基本概念

但是问题是：但它们之间的定量关系 V=IR 在不同条件下有不同含义，比如线性与非线性元件的区别

所以这一段要解决：深入理解 V=IR 的物理意义，掌握线性电阻和非线性元件的分析方法

V=IR 的三种形式

欧姆定律可写成 V=IR（求电压）、I=V/R（求电流）、R=V/I（求电阻）。三种形式等价但分别对应不同分析场景。线性电阻满足此关系，非线性元件（二极管等）不满足。

电阻与电导

电阻 R 单位欧姆(Ω)，电导 G=1/R 单位西门子(S)。并联分析中用电导更方便——G_total = G1+G2+...。串联分析中用电阻更方便——R_total = R1+R2+...

功率与焦耳定律

P = VI = I²R = V²/R（焦耳定律）。功率以热的形式耗散，选择电阻时要考虑功率额定值。I²R 形式在已知电流时最方便。

Concept 2

串联与并联：电路的基本拓扑

已经知道：你已经能区分串联和并联的连接方式

但是问题是：但串联和并联的电压、电流、电阻规律完全不同，混合电路的分析需要系统方法

所以这一段要解决：掌握串并联的等效电阻计算和分压分流原理

串联电路三规律

①电流处处相等 I₁=I₂=...；②总电压=各电压之和 V=V₁+V₂+...；③总电阻=各电阻之和 R=R₁+R₂+...。分压公式：Vx=(Rx/R_total)×V_total。

并联电路三规律

①各支路电压相等 V₁=V₂=...；②总电流=各支路电流之和 I=I₁+I₂+...；③1/R=1/R₁+1/R₂+...。分流公式：Ix=(R_other/(R₁+R₂))×I_total（两支路时）。

混联电路化简策略

从最内层开始：先合并纯并联组→再合并纯串联组→重复直至单一等效电阻。然后用分压/分流公式反向展开，求出每个元件上的电压和电流。

Concept 3

基尔霍夫定律：复杂电路的系统解法

已经知道：你已经能用欧姆定律分析简单串并联电路

但是问题是：当电路有多个独立回路和节点时，仅靠欧姆定律和串并联化简不够用——需要更高阶的定律

所以这一段要解决：学习基尔霍夫电流定律(KCL)和电压定律(KVL)，建立系统化的电路方程求解方法

KCL 电流定律

在任一节点，ΣI_in=ΣI_out，即 ΣI=0（流入正流出负）。本质是电荷守恒——节点不积累电荷。应用时标注各支路电流方向，方向可任意假设（解出来为负即反向）。

KVL 电压定律

沿闭合回路绕行一周，ΣV_rise=ΣV_drop，即 ΣV=0。本质是能量守恒——单位电荷绕回路一周电势能净变化为零。选定绕行方向后，电压升取正、电压降取负。

列方程四步法

①标出所有支路电流方向（未知可假设）；②对 n-1 个独立节点列 KCL；③对 b-(n-1) 个独立回路列 KVL；④联立方程组求解。b=支路数，n=节点数。

Summary

本课总结：直流电路的三大支柱

1.欧姆定律 V=IR 是电路分析的基石，适用于线性电阻元件
2.串联分压(电流相同)、并联分流(电压相同)，混联电路逐步化简
3.基尔霍夫定律(KCL+KVL)是复杂电路的系统解法：列方程→联立求解
4.KCL本质是电荷守恒，KVL本质是能量守恒
5.n节点 → n-1个独立KCL方程；b-(n-1)个独立KVL方程
