# 埃及分数：打开分数拆解之门

## 一句话钩子

**你以为分数只能是 1/7，但古埃及人告诉你：任何分数都可以拆成几个"1/n"的和。而真正好玩的是——拉马努金、高斯、爱因斯坦，三个人会用完全不同的方式拆它。**

---

## 问题

把 1/7 拆成三个不同的单位分数（分子为 1 的分数）之和：

```
1/7 = 1/a + 1/b + 1/c    （a, b, c 为正整数且互不相同）
```

---

## 三大解法（三种思维模式）

### 拉马努金法：通用公式流

**核心思维**：找到一个通用公式，批量生产解。

步骤：
1. 两边乘以 n，得到 1 = n/a + n/b + n/c
2. 选三个数使其和为 n，令 n/a = 1/k, n/b = 2/k, n/c = 3/k
3. 通常取 k = n，计算 a = n×(1+2+3)/1, b = n×(1+2+3)/2, c = n×(1+2+3)/3

对于 1/7：取 1+2+3=6 ≠ 7 → 换 1+2+4=7
得 (49, 24.5, 12.25) → 24.5 不是整数，舍去

换思路：保持 1+2+3=6，a = 7×6/1 = 42, b = 7×6/2 = 21, c = 7×6/3 = 14

**解：(42, 21, 14)** ✅

> 拉马努金的脑子：不是针对这一个问题，而是找到"所有这类问题"的通法。

---

### 高斯法：构造流

**核心思维**：我不需要从头算——我已经有一个等于 1 的漂亮等式，把它变形成目标就行。

步骤：
1. 已知：1 = 1/2 + 1/3 + 1/6（一个经典的恒等式）
2. 两边同乘 1/7 → 1/7 = 1/14 + 1/21 + 1/42

**解：(14, 21, 42)** ✅

> 高斯的脑子：数学不是从 0 开始推——而是把已知的好结构"平移"到新问题上。

---

### 爱因斯坦法：暴力枚举流（不推荐手算）

**核心思维**：把 1/7 化成循环小数 0.142857...，然后穷举所有可能组合。

理论上可行，但计算量巨大，是计算机的思路——不是人脑的优雅。

> 爱因斯坦的脑子（在这个问题上）："我不需要漂亮，我只需要对。"——但手算不是这么玩的。

---

## 更多通用技巧

### 裂项法（Telescoping）— 最优雅

核心公式：**1/[x(x+1)] = 1/x − 1/(x+1)**

举例：1/7 = 1/8 + 1/56 + 1/72
- 1/7 − 1/8 = 1/56
- 1/8 − 1/9 = 1/72
- ∴ 1/7 = 1/8 + 1/56 + 1/72

> 明明在拆分数，却感觉像在做拼图——每个碎片恰好碰到下一个。

---

### 公倍数法

1/3 = 1/a + 1/b

分子分母同乘 4：4/12 = (1+3)/12 = 1/12 + 3/12 = 1/12 + 1/4

**解：(12, 4)**

> 不是"算"出来的，是"看见"了 4 = 1+3 这个巧合。

## 为什么这个题让人上瘾

1. **同一问题、三种人格**：拉马努金的万能公式、高斯的构造平移、爱因斯坦的暴力穷举——你看到的不只是解法，是三种世界观
2. **手算能验证**：不像高等数学需要复杂工具，纸笔就能跟着做
3. **优雅感**：裂项法让人感觉"数学原来这么美"
4. **有挑战题**：解完马上想试下一个
