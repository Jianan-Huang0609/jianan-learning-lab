🔄 学习闭环

  1. 前测 Pre-test — 进入课件先诊断，定位当前水平
  2. 问题锚点 — 真实问题驱动动机（"RAG 怎么解决幻觉？""猫和狗的词义怎么比？"）
  3. 互动讲解 — Canvas 流程图、向量计算器、代码片段，不是看而是动手
  4. 三层脚手架 — 完整示范 → 部分提示 → 独立完成
  5. 概念测验 — 每题诊断错误原因，指出"错在哪、为什么、下一步"
  6. 后测 Post-test — 量化对比前测，验证学习增益
  7. AI 学伴 — 随时提问，不离开课件
