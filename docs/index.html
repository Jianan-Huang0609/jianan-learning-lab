🔄 学习闭环

  1. 前测 (Pre-test) — 进入课件先做诊断题，定位当前水平
  2. 问题锚点 — 用真实问题驱动学习动机（"为什么沉浮不同？""RAG怎么解决幻觉？"）
  3. 互动讲解 — 拖拽、仿真、动画，不是看视频而是动手操作
  4. 三层脚手架 — 完整示范 → 部分提示 → 独立完成，逐步放手
  5. 概念测验 — 每道题诊断错误原因，指出"错在哪、为什么、下一步"
  6. 后测 (Post-test) — 对比前测，量化学习增益
  7. AI 学伴 — 随时提问，不离开课件页面
